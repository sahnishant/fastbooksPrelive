# Fastbooks SEO Starter

Generated static site starter for Fastbooks prelaunch SEO.

## Replace before deploy

In `assets/site.js`, replace:

```js
const OWNER_WHATSAPP_NUMBER = "91XXXXXXXXXX";
```

with your WhatsApp number in international format without `+`.

Example:

```js
const OWNER_WHATSAPP_NUMBER = "919471499233";
```

## SEO notes

- No fake ratings are included in schema.
- Schema uses `Organization`, `WebSite`, `SoftwareApplication`, and page-level `WebPage`/`WebApplication`.
- Footer links create crawlable internal navigation.
- Expand thin placeholder resource pages before submitting them aggressively for indexing.


## Simplified WhatsApp early-access flow

The primary CTA now opens WhatsApp immediately with no required form fields.

Optional context fields are:
- Name
- Mobile number
- Email ID
- What do you want from Fastbooks?

All fields are optional. This is intentional because WhatsApp already gives you the user's sender number when they message you.
